# rollercoaster_ticket_calculator
A Python project that calculates rollercoaster ticket prices based on height, age, and optional photo selection.


🎢 Rollercoaster Ticket Calculator

A simple Python program that determines whether a user is eligible to ride a rollercoaster and calculates the total ticket price based on their age and optional ride photo selection.

Features

- Checks the minimum height requirement.
- Calculates ticket price based on age.
- Offers a free ride for visitors aged 45–55.
- Optionally adds the cost of a ride photo.
- Validates photo selection input ("y" or "n") using a loop.
- Displays the final bill.

Concepts Practiced

- Variables
- User Input
- Type Conversion ("int()")
- Conditional Statements ("if", "elif", "else")
- Nested Conditions
- "while" Loops
- "break" Statement
- String Methods (".lower()")
- Formatted Strings (f-strings)

How to Run

1. Make sure Python 3 is installed.
2. Clone this repository or download the source code.
3. Open a terminal in the project folder.
4. Run the program:

python rollercoaster_ticket_calculator.py

Welcome to the RollerCoastahhh!
